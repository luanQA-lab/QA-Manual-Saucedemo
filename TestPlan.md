Introduccion: 
    Este test plan cubre las pruebas de la funcionalidad principal del sitio Saucedemo:
        -Login
        -Listado de Productos
        -Carrito de Compra
        -Checkout

Objetivos de Prueba 
    Verificar que el login funciona correctamente con credenciales validas
    Validar que los productos se muestran correctamente
    Confirmar que el usuario puede completar una compra sin errores

Scope 
    Pruebas de Login  
    Lista de Productos
    Carrito de Compras
    Checkout

Fuera del Scope 
    Pruebas de Performance
    Pruebas de Api
    Pruebas de UI 

Estrategia de Pruebas   
    Tipo: Funcionales
    Tecnica: pruebas exploratorias + casos de pruebas

Ambiente de Pruebas
    Navegador: Edge
    Ambiente: https://www.saucedemo.com
    Datos: usuarios, productos

Criterios de Entrada
    Ambiente Funcionando
    Datos de prueba disponible(usuarios)

Criterios de Salida
    Todos los test ejecutados
    Defectos Criticos resueltos
    Reporte final generado


Gestion de Defectos
    Herramientas: Jira 


Cronograma
    Preparacion: 1 dia
    Ejecucion: 1 dia
    Reporte: 1 dia
